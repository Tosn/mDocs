# 手动验收清单（Acceptance Checklist）

> 配套：`feature/spec.md`（验收标准来源）、`feature/tasks.md`（R1–R3）。
> 自动化测试（202 个）已全绿，覆盖各模块逻辑；本清单是**需真机交互**的端到端验收，请在本机执行后逐项打勾。
> 勾选规则：`⬜` 未验 / `✅` 通过 / `❌` 失败（失败请在行尾记录现象，我据此修复）。

## 准备

1. 安装依赖并启动：
   ```bash
   rtk pnpm install
   rtk pnpm dev
   ```
2. 进入「设置」配置一个可用模型 + API Key（问答/嵌入需要联网与真实 Key）。
3. 准备若干测试文档：几个 `.md` / `.txt` / `.pdf`，一个含子目录的文件夹，一个普通网页 URL，一个需登录的网页 URL。

## R1 · 用户故事（spec 第 5 章）

### 史诗 A 文档入库
- ⬜ A1 上传单个/多个 md/txt/pdf → 出现在库中且可预览。
- ⬜ A1 同一上传入口选择**文件夹** → 连同内容按层级入库。
- ⬜ A1 选不支持格式 → 提示「不支持该格式」，文件夹上传时跳过并汇总。
- ⬜ A1 同名冲突 → 提示「保留两者 / 覆盖 / 取消」。
- ⬜ A2 文件夹批量导入 → 成功 N / 跳过 M 汇总；大量时有进度且可取消。
- ⬜ A3 应用内「新建」文档并写内容保存 → 可预览/搜索/问答。
- ⬜ A4 粘贴普通网页链接 → 抓取正文转 md 入库，保留图片与原链接。
- ⬜ A4 链接无效/正文为空 → 明确失败提示，不落空文档。

### 史诗 B 目录与文件管理
- ⬜ B1 新建多级目录、展开/折叠。
- ⬜ B2 重命名文件/文件夹；重名或非法名被拒绝。
- ⬜ B3 删除 → 二次确认 → 入回收站；回收站可恢复（含原父级缺失落根提示）。
- ⬜ B3 回收站项目可主动彻底删除（二次确认）。

### 史诗 C 预览与编辑
- ⬜ C1 md 渲染排版 / txt 文本 / pdf 翻页；长文滚动顺畅。
- ⬜ C2 预览中编辑 md/txt 并保存 → 重开为最新；编辑后搜索/问答同步。
- ⬜ C2 有未保存更改离开 → 「是否保存」拦截。
- ⬜ C2 pdf → 明确「不支持编辑正文」。

### 史诗 D 搜索
- ⬜ D1 关键词搜索文件名/内容 → 结果列表、命中跳转、空状态。
- ⬜ D1 断网下搜索仍可用。

### 史诗 E AI 问答
- ⬜ E1 跨文档库提问 → 基于文档内容作答；连续追问带上下文。
- ⬜ E2 回答标注来源、点击跳转到对应文档位置。
- ⬜ E3 文档无相关内容 → 明确「未在文档中找到相关内容」，不编造。
- ⬜ E4 设置中选模型、填 Key；重启后默认沿用上次模型；切换模型弹 Key 框并回填掩码。
- ⬜ E5 首次使用外部模型有隐私告知；设置中可随时查看。
- ⬜ E6 @文件 / @文件夹（递归）限定范围作答；不 @ 时默认全库。

## R2 · 边界情况（spec 第 6 章）

- ⬜ 空文件 / 损坏文件上传 → 提示并拒绝。
- ⬜ 超大文件上传 → 进度或大小提示，界面不卡死。
- ⬜ 批量导入中途关闭程序 → 重开状态一致，无半截损坏条目。
- ⬜ 删除非空文件夹 → 确认提示说明连同子项。
- ⬜ **回收站启动清理**：把某项 `purge_after` 调到过去 → 重启应用 → 该项被彻底删除且不再出现在回收站。
- ⬜ 恢复时原父级已删 → 落默认位置并告知。
- ⬜ 同一文档多处编辑 → 不互相覆盖、最终一致。
- ⬜ 空文档库提问 → 提示「请先添加文档」。
- ⬜ 模型未配置 / 超时 → 可读错误与重试入口，不卡死对话。
- ⬜ **无网络降级**：断网后查看/目录增删改/关键词搜索照常；仅 AI 问答不可用且有明确提示横幅。
- ⬜ @范围失效（被 @ 对象已删除/为空）→ 排除或提示「范围为空」，不静默改用全库。

## R3 · 两个新增需求

- ⬜ **@范围问答**：@ 一个文件夹（含子目录文档）提问，确认仅在该范围内检索；token 消耗相较全库下降（可观察响应/引用范围）。
- ⬜ **登录网页爬取**：粘贴需登录的 URL → 弹出内置浏览器窗口 → 登录并导航到目标页 → 点「抓取当前页」→ 正文转 md 入库（含图片、原链接）。
- ⬜ 登录爬取中途关闭窗口 / 取消 → 提示「已取消」，不落空文档。
- ⬜ 登录会话不被持久化（关闭后再开需重新登录），确认未额外留存凭据。
